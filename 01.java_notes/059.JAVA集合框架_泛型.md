## 1. 泛型的概念

### 1.1. 泛型的必要性
- 集合中能够存储任意类型的数据, 但取出来的时候都是 `Object` 类型的,  
  此时需要进行强制类型转换, 才能将元素恢复成原来的类型.

- 在实际使用中, 集合存储的元素的类型必须规定统一, 否则会出现混乱.
