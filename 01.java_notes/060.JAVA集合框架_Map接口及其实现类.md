## 1. Map 接口及其实现类
- `Map` 是一种映射关系的集合, 它将键映射到值的对象.   
  一个映射不能包含重复的键; 每个键最多只能映射到一个值.  

- `Map` 接口提供三种 `collection` 视图, 允许以键集, 值集  
  或键-值映射关系集的形式查看某个映射的内容.

- 严格意义上来说 `Map` 并非集合, 而是两个集合间的映射关系  
  `Map` 并没有继承于 `Collection` 接口, 它本身就是顶层接口.
